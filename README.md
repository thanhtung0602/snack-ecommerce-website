# 🍔 Website Bán Đồ Ăn Vặt (Snack E-commerce Website)

Dự án website thương mại điện tử mini được xây dựng bằng Core PHP, tập trung vào trải nghiệm mua sắm mượt mà và hệ thống quản trị nội dung (CMS) cơ bản.

## 🛠 Tech Stack (Công nghệ cốt lõi)
- **Frontend:** HTML, CSS, JavaScript (UI/UX thân thiện).
- **Backend:** Core PHP (Xử lý logic, Session/Cookies giỏ hàng).
- **Database:** MySQL (Thiết kế Schema tối ưu cho bán hàng).
- **Thiết kế:** Lên ý tưởng qua Wireframe.

## ✨ Chức năng nổi bật (Key Features)
- **Khách hàng:** Đăng ký/Đăng nhập, Duyệt sản phẩm theo danh mục, Thêm vào giỏ hàng, Đặt hàng và Xem lịch sử giao dịch.
- **Quản trị (Admin):** Bảng điều khiển quản lý toàn bộ Sản phẩm, Danh mục, Đơn hàng.
- **Database:** Tệp CSDL `qlbh.sql` đi kèm chứa cấu trúc bảng chặt chẽ và dữ liệu mẫu có sẵn để test luồng mua hàng.

---
*Dự án giúp mình củng cố vững chắc nền tảng PHP, cách tương tác với MySQL và xử lý các luồng logic của một website bán hàng thực tế.*
